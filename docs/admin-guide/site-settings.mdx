## Overview

The Settings page lets you configure site-wide options:
- Contact information displayed to customers
- Banner messages and branding
- Language preferences
- Import history and rollback

---

## Accessing Settings

1. Go to `/admin`
2. Click the **Settings** (gear icon) in the header
3. Or go directly to `/admin/settings`

---

## Contact Information

Update the contact details shown on the public site.

### Editable Fields

| Field | Description |
|-------|-------------|
| **Phone** | Main phone number |
| **Email** | Contact email address |
| **Address** | Physical address |
| **Hours** | Business hours |

### How to Update

1. Find the **Contact Information** section
2. Edit the fields you want to change
3. Click **Save** to apply changes

Changes appear immediately on the public site.

---

## Branding

Customize the appearance of your site.

### Banner Settings

The banner appears at the top of the public search page.

| Setting | Description |
|---------|-------------|
| **Banner Text** | Message to display |
| **Banner Color** | Background color |
| **Show Banner** | Toggle banner on/off |

### How to Update Banners

1. Find the **Branding** section
2. Enter your banner message
3. Choose a color
4. Toggle visibility
5. Click **Save**

---

## Language Settings

Switch between English and Spanish for the admin interface.

1. Find the **Language** toggle at the top of Settings
2. Click to switch between `EN` and `ES`
3. The interface updates immediately

This only affects the admin dashboard. The public site has its own language toggle.

---

## Import History & Rollback

View recent data imports and undo them if needed.

### Viewing Import History

The Import History section shows:
- Date and time of import
- Number of records added/updated/deleted
- Import status (success/failed)

### Rolling Back an Import

If you need to undo a recent import:

1. Find the import in the history list
2. Click **Rollback**
3. Confirm the rollback
4. Wait for the restore to complete

**Important:** You can only rollback the most recent import. Older imports cannot be rolled back individually.

---

## Logging Out

To log out of the admin dashboard:

1. Click the **Logout** button in Settings
2. You'll be redirected to the public search page
3. You'll need to re-enter the password to access admin again
