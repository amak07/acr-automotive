# Getting Started

Welcome to **ACR Automotive**, an auto parts cross-reference search platform built for parts counter staff and automotive professionals.

## What is ACR Automotive?

ACR Automotive helps you:
- **Search parts** by SKU, OEM number, or description
- **Find cross-references** between different part manufacturers
- **Manage inventory** through Excel import/export
- **View part images** including 360° views
