## Prerequisites

Before you begin, make sure you have:

- [ ] **Node.js 18+** - [Download](https://nodejs.org/)
- [ ] **Docker Desktop** - [Download](https://www.docker.com/products/docker-desktop/) (for local Supabase)
- [ ] **Git** - [Download](https://git-scm.com/)

<Callout type="info">
Docker Desktop must be running before you start. Open it and wait for it to fully load.
</Callout>

---

## Step 1: Clone the Repository

```bash
git clone https://github.com/amak07/acr-automotive.git
cd acr-automotive
```

**What you should see**: A new `acr-automotive` folder with the project files.

---

## Step 2: Install Dependencies

```bash
npm install
```

**What you should see**: Progress bars as packages download. This takes 1-2 minutes.

<Callout type="warn">
If you see permission errors, try running your terminal as administrator.
</Callout>

---

## Step 3: Set Up Environment

```bash
cp .env.example .env.local
```

Now edit `.env.local` with these values for local development:

```bash
# .env.local
NEXT_PUBLIC_SUPABASE_URL=http://localhost:54321
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZS1kZW1vIiwicm9sZSI6ImFub24iLCJleHAiOjE5ODM4MTI5OTZ9.CRXP1A7WOeoJeXxjNni43kdQwgnWNReilDMblYTn_I0
ADMIN_PASSWORD=<your-secure-password>
NEXT_PUBLIC_ENABLE_POST_MVP=true
```

<Callout type="info">
The anon key above is the **default local Supabase demo key** - safe for development only. It only works with your local Docker instance and should never be used in production.
</Callout>

<Callout type="warn">
**Security**: Set a strong password for `ADMIN_PASSWORD`. This protects your admin panel and documentation access.
</Callout>

## Step 4: Start Local Supabase

```bash
npm run supabase:start
```

**What you should see**: Docker images downloading (first time only), then services starting.

```

         API URL: http://localhost:54321
     GraphQL URL: http://localhost:54321/graphql/v1
          DB URL: postgresql://postgres:postgres@localhost:54322/postgres
      Studio URL: http://localhost:54323
    Inbucket URL: http://localhost:54324
```

<Callout type="warn">
First-time setup downloads ~500MB of Docker images. This takes 2-5 minutes depending on your internet.
</Callout>

## Common Issues

### Docker not running

**Error**: `Cannot connect to Docker daemon`

**Fix**: Open Docker Desktop and wait for it to fully start before running `npm run supabase:start`.

---

### Port already in use

**Error**: `Port 3000 is already in use`

**Fix**: Either close the other application using port 3000, or run:

```bash
npm run dev -- -p 3001
```

---

### Supabase won't start

**Error**: Various Docker errors

**Fix**: Try resetting:

```bash
npm run supabase:stop
npm run supabase:start
```

| `npm run supabase:stop` | Stop local database |
